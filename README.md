
# Banking Management System

A console-based banking application built in Java.

## Features
- Savings and Current Account support
- Deposit and Withdrawal
- Overdraft protection for Current Account
- Mini Statement (last 5 transactions)
- Balance enquiry

## OOP Concepts Used
- Abstraction (Account abstract class)
- Inheritance (SavingsAccount, CurrentAccount)
- Polymorphism (Withdraw behaves differently per account)
- Encapsulation (Private fields with getters)

## How to Run
Open in IntelliJ → Run Main.java
